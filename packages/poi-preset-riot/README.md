# poi-preset-riot

Use [Riot](http://riotjs.com/) with Poi.

## Install

```bash
yarn add poi-preset-riot --dev
```

## Usage

Activate it in config file:

```js
// poi.config.js
module.exports = {
  presets: [
    require('poi-preset-riot')()
  ]
}
```
